# Приложение для создания и редактирования информации о встречах сотрудников

Написано для Node.js 8 и использует библиотеки:
* express
* sequelize
* graphql

## Команды
Установка зависимостей:
```
npm install
```
Запуск:
```
npm run dev
```

Сброс данных в базе:
```
npm run reset-db
```

Запуск теста:
```
npm test
```

Запуск линтера:
```
npm run lint
```
